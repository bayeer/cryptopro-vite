# How to use CryptoPro browser extension in JavasScript

1. Clone this repository:
```bash
git clone git@github.com:bayeer/cryptopro-vite.git
```

2. Install libraries:
```bash
yarn
```

3. Build the project:
```bash
yarn run dev
```

4. Open browser:
https://localhost:5173

5. Open browser development tools (F12)
